# AliYil.Mandelbrot
Multi-core Mandelbrot set visualizer / explorer written in .net

## Controls

| Key(s) | Action |
| :- | :- |
| Arrow keys | Move camera |
| + (plus) / - (minus) | Zoom in / out |
| Space | Reset camera |
| W / Q | Increase / decrease max. iterations |
| X / Z | Increase / decrease color palette offset |
